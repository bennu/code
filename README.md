# Bennu Code

Generador de proyectos fullstack listos para producción. Elige tu stack y descarga al instante.

## Stack

- [Next.js 15](https://nextjs.org/) — App Router
- [MUI (Material UI)](https://mui.com/) — componentes y theming
- [JSZip](https://stuk.github.io/jszip/) — generación de ZIPs en el cliente
- [Anime.js](https://animejs.com/) — animaciones

## Desarrollo

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
pnpm start
```
